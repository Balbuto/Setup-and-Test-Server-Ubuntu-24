# Changelog

## [2] – 2026-05-20

### Исправлено
- Устранено дублирование параметров BBR в файле `/etc/sysctl.d/99-xray-highload.conf` (параметры оставлены только в шаге 4).
- Исправлено логирование: теперь файл создаётся в `/var/log/server-setup.log` с fallback на `/tmp` или `$HOME`.
- Добавлена проверка дистрибутива (Ubuntu 24.04 рекомендуется).

### Изменено
- Упрощена структура меню: базовая настройка теперь одним пунктом.
- В мультитест интегрированы все тесты (14 штук) – больше не требуется скачивать внешние скрипты.
- Docker после установки автоматически запускается (`systemctl enable docker --now`).

### Добавлено
- Функция `init_logging()` для безопасной инициализации лог-файла.
- Поддержка фиктивной `SSH_CLIENT` для совместимости с `ufw-manager.sh`.
- Возможность запуска мультитеста без прав root.

## [1.0.2] – 2026-05-15

- Первая стабильная версия с двумя основными разделами: настройка и тестирование.
- Интеграция `ufw-manager` и `ipv6-manager`.
- Включение BBR, отключение IPv6, установка Docker.

## [1.0.1] – 2026-05-10

- Добавлено меню тестов (censorcheck, bench.sh, YABS, IP Region, Instagram Audio).
- Исправлены ошибки с переменной окружения.

## [1.0] – 2026-05-01

- Начальная версия: обновление системы, BBR, запуск ipv6-manager и ufw-manager.
